# Lead Processing & Matching Automation System

Production-grade automation system for processing, enriching, and routing inbound leads using API-driven workflows.

## Impact

- Reduced lead response time by **75%**
- Decreased manual workload by **67%**
- Generated **46% of inbound sales pipeline**

- ## System Architecture

- Orchestration: n8n workflows  
- CRM: Salesforce  
- Data Processing: JavaScript logic  
- External APIs: Data enrichment services (Clay)

- ## Key Features

- Automated lead ingestion and validation  
- Fuzzy matching (Jaro-Winkler) for deduplication  
- Multi-stage routing logic (auto / triage / net-new)  
- Real-time and batch processing pipelines
